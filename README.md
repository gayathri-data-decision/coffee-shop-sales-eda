# Coffee Shop Sales – Exploratory Data Analysis

## Project Overview
This project performs exploratory data analysis (EDA) on a coffee shop sales dataset obtained from Kaggle. 
The goal is to understand sales trends, customer behavior, and product performance using Python.

## Dataset
- Source: Kaggle
- Data includes transaction details, product categories, dates, and revenue.

## Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Key Analysis Performed
- Data cleaning and preprocessing
- Sales trends over time
- Product-wise performance
- Revenue analysis

## Status
Completed EDA. Further improvements and visualizations may be added.
