# Coffee Shop Sales – Exploratory Data Analysis

## Overview
This project performs Exploratory Data Analysis (EDA) on a real-world coffee shop sales dataset to understand customer purchasing behavior, peak sales periods, and revenue patterns.

The dataset contains 3,547 transaction records with information on coffee type, time of purchase, weekday, month, and transaction value.

## Objectives
- Identify peak sales hours and busiest weekdays
- Analyze best-selling and revenue-generating coffee types
- Examine time-of-day and monthly sales patterns
- Support data-driven staffing and inventory decisions

## Dataset
- Source: Kaggle (Coffee Shop Sales Dataset)
- Records: 3,547 transactions
- Note: Dataset is not uploaded due to redistribution restrictions

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Key Insights
- Morning hours show the highest sales volume, with peak activity around 10 AM
- Tuesday records the highest number of transactions, while Sunday shows the lowest
- Americano with Milk is the most frequently purchased coffee
- Latte generates the highest total revenue
- March shows the highest monthly sales volume
